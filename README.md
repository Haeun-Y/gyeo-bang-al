## ⛄ 스터디 진행 방식
* 스터디 일시 : 매주 토요일 오전 9시 30분 - 12시 30분
* 스터디 장소 : 공대 도서관 모둠학습실
* 지난 주차에서 어려웠던 문제 공유 및 질문 + 해당 주차 개별적으로 공부
* 다음 스터디 날짜인 토요일 오전 9시 30분전까지 해당 주차 내용 + 필수 문제 풀어서 pull request로 과제 제출
* 필수 문제는 바킹독 문제집 리드미의 ✔ 문제
* pull request 형식 : [이름] 주제 (ex. [윤하은] 스택, 큐, 덱, 스택의 활용)
